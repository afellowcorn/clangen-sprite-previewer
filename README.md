\# clangen-sprite-previewer
A tool for ClanGen modding that allows you to preview generated sprites.

\# Usage
Replace the sprites folder with yours, making sure to include the dicts folder.
If your mod changes sprite generation you have to to modify `display\_sprites.py` and add any needed imports if necessary.

Refer to [ClanGen's](https://github.com/ClanGenOfficial/clangen) README for instructions on running from source.
